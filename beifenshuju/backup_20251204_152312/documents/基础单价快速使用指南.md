# 基础单价快速使用指南

## 🎯 什么是基础单价？

**基础单价 = 采购单价 ÷ 采购转化率**

**作用：** 统一不同采购单位的价格，方便直接对比

---

## 📖 使用示例

### 为什么需要基础单价？

**问题场景：**
```
物料A：100元/箱（1箱=10个）
物料B：12元/个

哪个更便宜？ 🤔
```

**解决方案：**
```
物料A基础单价：100 ÷ 10 = 10元/个
物料B基础单价：12 ÷ 1 = 12元/个

结论：物料A更便宜！ ✅
```

## 🖥️ 界面位置

### 编辑物料页面

```
产品物料库 
  → 选择物料 
  → 点击编辑 
  → 切换到"采购属性"标签
  → 查看"基础单价"字段
```

**显示效果：**
```
┌────────────────────────────────┐
│ 采购单价：¥100.00              │
│ 采购转化率：10                 │
│ 基础单价：¥10.00 [只读]       │
│ 计算公式：采购单价÷采购转化率  │
└────────────────────────────────┘
```

---

### 查看物料页面

```
产品物料库 
  → 选择物料 
  → 点击查看 
  → 切换到"采购属性"标签
  → 查看"基础单价"（绿色高亮）
```

---

## ✨ 特点

1. ✅ **自动计算** - 修改采购单价或转化率时自动更新
2. ✅ **只读字段** - 不能手动修改，避免错误
3. ✅ **实时显示** - 立即看到计算结果
4. ✅ **自动保存** - 保存物料时自动保存基础单价

---

## 💡 常见问题

**Q: 基础单价显示0.00？**
```
A: 检查采购单价是否为0，或采购转化率是否正确
```

**Q: 能手动修改基础单价吗？**
```
A: 不能，基础单价由系统自动计算
   如需修改，请修改采购单价或采购转化率
```

**Q: 修改后没看到变化？**
```
A: 页面会实时更新，保存后写入数据库
```
