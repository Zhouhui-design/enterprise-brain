## 🔧 技术实现

### 菜单配置
所有菜单都配置在 `/07-frontend/src/layout/index.vue` 的 `loadMenus()` 方法中。

### 路由映射
每个菜单项都对应一个前端页面路径：

| 菜单名称 | 页面路径 |
|---------|---------|
| 主生产计划 | `/pages/production-planning/ProductionPlanList.vue` |
| 主生产计划调整 | `/pages/production-planning/PlanAdjustment.vue` |
| 产能规划 | `/pages/production-planning/CapacityPlanning.vue` |
| 财务概览 | `/pages/finance/FinanceOverview.vue` |
| 成本管理 | `/pages/finance/CostManagement.vue` |
| ... | ... |

### 图标配置
为每个菜单配置了合适的Element Plus图标：
- 📊 财务相关：`el-icon-s-finance`, `el-icon-coin`, `el-icon-wallet`
- 📝 文档相关：`el-icon-document`, `el-icon-document-add`
- 📈 分析相关：`el-icon-data-analysis`, `el-icon-data-board`
- 👤 用户相关：`el-icon-user`, `el-icon-office-building`

```

```
