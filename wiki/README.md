## 🎯 使用指南

### 1. 当出现BUG时

1. **先查看数据流规则**：确认业务规则是什么
2. **查看字段映射关系**：确认字段对应关系
3. **使用调试指南**：按步骤排查问题
4. **查看常见问题**：看是否有类似案例

### 2. 开发新功能时

1. 先在Wiki中记录数据流规则
2. 记录字段映射关系
3. 实现功能
4. 如有问题，按调试指南排查

### 3. 更新Wiki

- 发现新的数据流规则时，立即记录
- 修复BUG后，更新到常见问题
- 发现调试技巧时，更新调试指南
